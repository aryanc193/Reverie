# Agent Onboarding Guide

## Objective
Enable autonomous AI agents to contribute to Reverie.

## Before Starting
Read files in this order:

1. 00_PROJECT_OVERVIEW.md
2. 01_PRODUCT_REQUIREMENTS.md
3. 02_BACKEND_ARCHITECTURE.md
4. 03_AUTH_SYSTEM_SPEC.md
5. 04_DATABASE_SCHEMA_ROADMAP.md
6. 05_AI_SYSTEM_DESIGN.md
7. 06_ENGINEERING_TODO.md
8. 08_FRONTEND_WORKFLOW.md (frontend / Figma MCP work only)

## Expected Output From Agents
- Production-grade code
- Type-safe APIs where possible
- Modular architecture
- Documentation for all changes

## Coding Standards
- Separation of concerns
- Small services
- Reusable middleware
- Strong validation
- Testability


## Frontend Work

For UI implementation, read `08_FRONTEND_WORKFLOW.md`, fill in `figma-links.md`, authenticate Figma MCP, then run `/frontend`.
