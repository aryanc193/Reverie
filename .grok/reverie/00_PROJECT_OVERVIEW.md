# Reverie — Project Overview

## Vision
Reverie is an AI-first reflective companion designed to help users capture thoughts, journal experiences, build self-awareness, and receive intelligent insights from their personal history.

## Core Principles
- Privacy-first personal knowledge system
- AI-native user experience
- Long-term memory and reflection
- Structured + unstructured journaling
- Conversation-driven interactions

## Current Engineering Direction
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- Authentication: JWT-based auth
- API-first architecture
- Modular services and middleware

## Initial Milestones
1. Authentication system
2. User management
3. Memory/journal storage
4. AI reflection engine
5. Retrieval and search
6. Personalization layer
